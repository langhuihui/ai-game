MCP 工具配置

使用说明： 选择适合你的配置方式。推荐使用 stdio 模式，更稳定可靠。

SSE 配置 (推荐)

mcp.json 
{
  "mcpServers": {
    "game-server": {
      "url": "http://localhost:3000/mcp"
    }
  }
}

启动说明

1. 启动游戏服务器
npm start
2. 复制上面的配置到 mcp.json
3. 重启 Cursor 编辑器
优势： SSE 连接支持真正的多人联机，所有客户端共享同一个游戏世界！
注意： 确保游戏服务器正在运行，否则 MCP 工具将无法使用。
